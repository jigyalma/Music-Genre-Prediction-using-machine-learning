Music Genre Prediction Using Machine Learning

Problem Statement

The world seems incomplete without music, and with many different songs around the world, we have different genres of music such as electronic, rap, classical, hip-hop, rigsar, boedra, tsangmo, and many more. Genre is a category that identifies some pieces of music as it belongings to a shared tradition or set of conventions. Features that are used to predict the genre of a music are purely factual and are all are predicted in strict accordance with machine learning algorithm, the final result might be sometimes inaccurate.

Aims

The aim of this project is to build a platform which predicts music into its respective genre using machine learning.

Scope

The system can be used internationally and most probably it will be beneficial to the music industry, with the aim to classify the music genre.
